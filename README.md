## 🍽️ Tentang PiringKita!
**PiringKita!** adalah sebuah aplikasi web yang dirancang untuk membantu mahasiswa dan masyarakat di wilayah Kota dan Kabupaten Bandung dalam mencari rekomendasi tempat makan. Aplikasi ini menggunakan pendekatan *crowdsourced*, di mana pengguna dapat berkontribusi secara langsung dengan menambahkan tempat makan baru, memberikan ulasan, membagikan rating, serta menyimpan tempat makan favorit.

Proyek ini dibangun untuk memenuhi Tugas Besar mata kuliah **CAK3BAB3 - Implementasi dan Pengujian Perangkat Lunak (IMPAL)**.

## ✨ Fitur Utama
- **Autentikasi & Manajemen Akun**: Registrasi, Login (User & Admin), Reset Password, dan Kelola Profil Pengguna.
- **Eksplorasi Kuliner**: Cari dan filter tempat makan berdasarkan kriteria seperti harga, kategori, lokasi, atau jenis makanan.
- **Kontribusi Komunitas (*Crowdsourcing*)**: Pengguna dapat menambahkan data tempat makan baru (beserta foto dan pin lokasi) serta memperbaruinya.
- **Ulasan & Rating**: Berikan *review* dan rating (1-5 bintang) pada tempat makan. Pengguna juga dapat mengubah atau menghapus ulasan yang telah mereka buat.
- **Favorit & Rekomendasi**: Menyimpan tempat makan ke daftar favorit dan melihat rekomendasi tempat makan populer berdasarkan ulasan/rating tertinggi.
- **Integrasi Peta**: Menampilkan titik lokasi presisi tempat makan di peta interaktif (menggunakan Leaflet.js & OpenStreetMap).
- **Verifikasi Admin**: Sistem moderasi bagi Admin untuk menyetujui (*approve*) atau menolak (*reject*) penambahan tempat makan dari pengguna demi menjaga validitas informasi.

## 🛠️ Tech Stack (Teknologi yang Digunakan)
Aplikasi PiringKita! dikembangkan menggunakan arsitektur berbasis layanan (*service-oriented architecture*) dengan pembagian:
- **Frontend (Lapisan Presentasi)**: HTML, CSS, JavaScript
- **Backend (Lapisan Logika Bisnis)**: Java
- **Database (Lapisan Basis Data)**: MySQL / PostgreSQL
- **Layanan API**: Peta Interaktif (Leaflet.js / OpenStreetMap API)
